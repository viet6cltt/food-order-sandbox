# Food Delivery Web Project

# Installation

## Prerequisites

* Ensure you have installed Node.js and npm

## Steps

### Clone the Repository

```
git clone https://github.com/HieuSerend/Food-Delivery.git

cd Food-Delivery
```

### Setup Frontend
Navigate to the frontend folder and run:

```
npm install
npm run dev
```
Access the application at http://localhost:5173/
